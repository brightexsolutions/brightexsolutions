## 👋 About

Brightex Solutions is a software consultancy building modern web and mobile products for businesses worldwide. We take projects from idea to launch: architecture, design, engineering, payments, and deployment.

Behind the studio is **[Godwin Ochieng](https://github.com/gbrowns)**, a Senior Full-Stack Engineer (web & mobile) who has shipped enterprise SaaS, AI-driven platforms, and cross-platform apps used in production. Every Brightex project is built to a strict engineering standard: secure auth, real error handling, health monitoring, and CI/CD from day one.

---

## 🛠️ What We Build

- **AI-powered products** — LLM chat, natural-language workflows, and AI document generation with Claude & Gemini
- **Full-stack web apps** — Next.js platforms with Supabase / Firebase backends, RBAC, and dashboards
- **Cross-platform mobile** — React Native / Expo apps sharing a codebase with the web
- **Payments & integrations** — M-Pesa (Daraja), Stripe, SMS, email, and web push
- **Enterprise SaaS** — multi-tenant systems, reporting engines, and PDF/document pipelines
